# 🔐 TeleOTP-Auth

A professional, secure and open-source OTP authentication system for Telegram bots, built with Python.

Developed by **Amir Asgari**

---

TeleOTP-Auth is a lightweight and secure OTP (One-Time Password) authentication system designed specifically for Telegram bots.

It allows developers to easily add user registration and login verification to their bots without building an authentication system from scratch.

Users receive an OTP code for login or registration, and after verification, they gain access to the bot or service.

This project is fully open-source and designed for real-world, educational, and scalable usage.

## ⚙️ Installation / نصب

Clone repository:

git clone https://github.com/USERNAME/TeleOTP-Auth.git  
cd TeleOTP-Auth  

Install dependencies:

pip install -r requirements.txt  

Configure project (config.py or .env):

- Telegram Bot Token  
- Database credentials  
- OTP settings (expiry time, length, etc.)

Run project:

python main.py
